# Contributing to RTLDI Atlas

Thank you for your interest in contributing to the RTLDI Atlas project!

## How to Contribute

### Reporting Issues
- Use the GitHub Issues tab to report bugs, suggest improvements, or request new features.
- Please include as much detail as possible (data version, steps to reproduce, expected vs actual behavior).

### Suggesting Changes
- Fork the repository and create a feature branch from `main`.
- Make your changes.
- Submit a Pull Request with a clear description of what was changed and why.

### Running the Pipeline Locally
1. Clone the repository
2. Install dependencies (see README.md)
3. Download the required V-Dem and World Bank data
4. Run the build scripts in `scripts/`

### Code Style
- Keep changes focused and well-documented.
- Follow the existing structure in `src/` and `docs/`.

## Questions?
Open an issue or reach out via the project maintainer.

We appreciate contributions that help make this resource more accurate and useful for researchers and policymakers.
